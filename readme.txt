== Description ==

= In English =

This plugin displays some useful statistics from your <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> account. All in one page with filters.

All the data is stored in your wordpress database and updated every 180 minute by default (you can specify this in plugin settings).

The stat page (in admin backend) has the following filters:

*   Today and Yesterday
*   Monthly
*   Yearly
*   Total

The widget is included to let you show how rich you are whith <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> ;)

The widget has the following settings:

*   Show the number of sites
*   Show CY
*   Show Google PR
*   Show Yandexed pages
*   Show Googled pages
*   Show Author link

Create an account in <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> if you haven't done it yet and start earning money by selling links!

= По-русски =

Плагин показывает статистику вашего аккаунта в <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> на одной странице с фильтрами.

Все данные хранятся в базе данных Wordpress и обновляются каждые 180 минут (по умолчанию, вы можете изменить это значение в настройках плагина).

Страница статистики (в панели администратора) содержит следующие фильтры:

*   Сегодня и вчера
*   Месяц
*   Год
*   Всего

Чтобы похвалиться своим заработком в <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> можно активировать виджет;)

Виджет имеет следующие настройки:

*   Показывать кол-во сайтов
*   Показывать тИЦ
*   Показывать Google Page Rank
*   Показывать кол-во страниц в Яндексе
*   Показывать кол-во страниц в Google
*   Показывать ссылку на разработчика (ссылка заключена в теги <noindex></noindex> ;)

Создайте аккаунт в <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a>если ещё не сделали это и начните зарабатывать деньги на продаже ссылок!

== Installation ==

= In English =

1. Upload `wp-sape-stat` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Settings->WP Sape Stat` and configure the plugin
4. Check your <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> statistics in `Tools->WP Sape Stat`
5. Activate the widget and show how rich you are with <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> (if you wish)

= По-русски =

1. Загрузите папку `wp-sape-stat` в директорию `/wp-content/plugins/` вашего Wordpress
2. Активируйте плагин через меню 'Плагины' вашей админ-панели
3. Перейдите на страницу `Настройки->WP Sape Stat` и настройте плагин
4. Проверяйте свою статистику <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a> на странице `Инструменты->WP Sape Stat`
5. Если хотите, активируйте виджет, чтобы показать другим, насколько богатым делает вас <a href="http://www.sape.ru/r.eMotfFWAlO.php">Sape.ru</a>
